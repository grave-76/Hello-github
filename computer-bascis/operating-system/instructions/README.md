# Dos命令

```bash
# 鼠标右键粘贴

# 盘符切换
D:

# 查看当前目录下所有文件
dir

# 切换目录 change directory 注意斜杠与反斜杠
cd /d D:
cd /d D:\test

# 返回上一级
cd ..

# 清理屏幕 clear screen
cls

# 退出
exit

# 查看电脑ip
ipconfig

# 打开应用
calc 计算器
mspaint 画图
notepad 记事本

# ping 命令
ping www.baidu.com/ip

# 文件操作
md test 创建目录
cd>test.txt 创建文件
del test.txt 删除文件
rd test 删除目录
```
